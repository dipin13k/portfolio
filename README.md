# Dipin Khadka - Personal Portfolio

Welcome to my personal portfolio! This project showcases my skills, journey, and projects as an aspiring Web Developer and Web3 Enthusiast based in Bhaktapur, Nepal.

## 🚀 About the Project
This portfolio is built using modern web development technologies and features a fully interactive 3D character powered by WebGL and Three.js, complete with responsive animations and physics simulation.

### Key Features:
- **Interactive 3D Model**: Real-time rendering of a 3D avatar with mouse-tracking logic.
- **Scroll-Linked Animations**: Super smooth parallax and UI sweep animations using GSAP (GreenSock).
- **Responsive Design**: Designed beautifully for both Desktop and Mobile experiences.
- **Heavy Physics Engine**: React Three Fiber and Rapier powers the "Tech Stack" environment physics.

## 🛠 Tech Stack
- **React.js (Vite)**
- **Three.js** & **@react-three/fiber**
- **GSAP** (ScrollSmoother & ScrollTrigger)
- **TypeScript** & **Vanilla CSS**

## 💻 Running Locally
To run this project on your own machine:

1. Clone the repository 
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
